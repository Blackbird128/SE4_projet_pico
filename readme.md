# Projet Pico ordinateur Louis Bonningre & Antoine Lecomte

Lors de ce projet nous avons réalisé une carte mère qui interface avec plusieurs cartes filles développées par d'autres binômes

## Organisation du dépôt GIT :

+ Bootloader : ce dossier contient les bootloader des 328p et 8u2 ainsi qu'un programme blink qui nous a permis de tester nos Atmega
+ OS : ce dossier contient le système d'exploitation de notre carte avec l'ordonnanceur et les differentes fonctions (ISP, S7S, Serie...)
+ PicoMother : ce dossier contient le projet Kicad de notre carte mère*
+ PicoShield : ce dossier contient le projet Kicad du Shield pour Arduino et les differents codes des focntions que nous avons testées (S7S, Série, UART et SD)
